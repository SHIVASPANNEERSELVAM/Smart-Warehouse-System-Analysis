# Smart Warehouse Management System

A comprehensive Java-based Smart Warehouse Management System developed using Object-Oriented Programming (OOP), Data Structures, Algorithms, and Advanced Java concepts.

## Features

### Product Management
- Product Registration
- Inventory Tracking
- Category Management
- Quantity & Price Management

### Warehouse Layout
- 2D Array-based Warehouse Structure
- Product Placement Management
- Warehouse Analytics

### OOP Concepts
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- Interfaces

### Data Structures
- Binary Search Tree (BST)
- AVL Tree
- Graph (BFS & DFS)
- Stack
- Queue
- Priority Queue (Heap)
- HashMap
- TreeMap
- HashSet
- TreeSet

### Algorithms
- Binary Search
- Bubble Sort
- Merge Sort
- Quick Sort
- Recursion
- Dynamic Programming
- Backtracking

### Advanced Java
- Generics
- Exception Handling
- Custom Exceptions
- File Handling
- Serialization
- Multithreading
- Lambda Expressions
- Stream API
- Singleton Design Pattern

### Performance Monitoring
- Execution Time Calculation
- Memory Usage Calculation
